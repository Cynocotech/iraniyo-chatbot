🏢 Iranian Business Directory API

Connect your chatbot to the Iraniu Business Directory to help users find Iranian businesses, restaurants, hairdressers, shops, and services across the UK. Enable this for any agent with a simple toggle.

✨ Features:
• Search 100+ Iranian businesses in London, Manchester, Birmingham
• Categories: رستوران, آرایشگاه, فروشگاه مواد غذایی, and more
• Enable per-agent with category filtering
• Full business details with Google Maps links

🤖 Enable Directory for Agents

To enable Directory API access for an agent:

  1. Go to the 🤖 Agents tab
  2. Click ✏️ Edit on any agent (e.g., Dr. Yas)
  3. Find the "Directory API Access 🏢" dropdown
  4. Select "Enabled"
  5. (Optional) Select specific categories to restrict access
  6. Click 💾 Save Agent

📚 Documentation & API Endpoints

Public API Endpoints:
GET /api/directory/categories
GET /api/directory/businesses?category=X&city=Y
GET /api/directory/businesses/{slug}

Example Usage:
# Get all categories
curl /api/directory/categories

# Search restaurants in London
curl "/api/directory/businesses?category=رستوران&city=London"
